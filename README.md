# 🏥 Medicare Website

A simple and responsive healthcare website built using HTML, CSS, and JavaScript.

## 📌 Features

- Responsive Design
- Modern User Interface
- Smooth Navigation
- Healthcare Services Section
- About Section
- Contact Information
- Appointment Button with JavaScript Alert

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

